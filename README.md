# React ToDo WebApp

Simple ToDo web app using React and Tornado.

### Setup:

git clone https://github.com/mahakal/react-todo-app/
cd react-todo-app  
npm i  
pip3 install -r python_server/requirements.txt    

### Running:  

npm start
